# SmartQuote

**智能家居方案设计软件 / Professional Smart Home Solution Design Software**

专业的智能家居安装商、经销商方案设计工具，一站式完成点位图设计、3D预览、报价生成与方案输出。

---

## 核心功能

- **点位图设计** — 在户型图上自由放置设备图标，支持多楼层、PDF底图导入、墙体绘制、WiFi信号覆盖可视化
- **3D实景预览** — 基于Three.js的3D渲染引擎，实时预览照明效果，支持开关灯、调节亮度和色温
- **智能报价** — 从产品库快速选品，自动计算报价与成本分析，打印专业报价单
- **方案设计** — 多页幻灯片式方案展示，全屏演示模式，导出PPT/PDF/图片
- **产品管理** — 完整的产品数据库管理，支持Excel导入、自定义品牌和规格
- **数据安全** — 完全离线运行，数据本地存储，自动保存与项目备份
